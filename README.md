# Cardia - Intelligent Heart Disease Prediction System

Cardia is a modern web application designed to predict heart disease risk using Machine Learning. This repository contains the Front-end application built with React, TypeScript, and Tailwind CSS.

## Tech Stack
- **Framework:** React 19 & Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Routing:** React Router DOM v7
- **HTTP Client:** Axios
